# 🎂 İnteraktif Pasta ve Mum Üfleme Web Uygulaması

Bu proje, React ile geliştirilmiş interaktif bir web uygulamasıdır. Kullanıcı, doğum günü pastasını görüntüler, mumları üfleyerek söndürebilir ve kutlamayı görebilir. 🌟

---

## 🔹 Özellikler

- **Karşılama Ekranı:** Özel mesaj ve giriş animasyonu.
- **Dilek Tutma:** Basit bir countdown efekti ile dilek tutma aşaması.
- **Pasta Üfleme:** Mikrofon ile üfleme algılaması; mumlar söner ve duman animasyonu başlar.
- **Kutlama Ekranı:** Confetti efekti, animasyonlu mesajlar ve kutlama metinleri.
- **Responsive:** Hem mobil hem masaüstü uyumlu tasarım.
- **Tema Değişimi:** Mumlar üflendiğinde renk ve arka plan değişimi.

---

## 🔹 Kurulum ve Çalıştırma

1. Projeyi klonlayın:
   git clone <repo-url>
   cd <repo-adi>

2. Bağımlılıkları yükleyin
   npm install react-confetti
   npm install

3. Uygulamayı başlatın
   npm start
